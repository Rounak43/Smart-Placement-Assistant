Smart Placement Performance Platform

A modern web platform designed to help students track their placement preparation, practice DSA problems, and analyze their performance through structured roadmaps and analytics.
This project aims to make placement preparation organized, trackable, and data-driven.

🔗 GitHub Repository: https://github.com/Rounak43/Smart-Placement-Assistant

📌 Features
1 DSA Sheet Tracker
Track solved problems with checkboxes and monitor progress.

2 Placement Roadmap
Structured roadmap for coding interview preparation.

3 Aptitude Test Module
Timed tests with easy, medium, and hard difficulty levels.

4 Progress Analytics Dashboard
Visual insights into learning performance.

5 Company-wise Question Tracking
Shows which companies commonly ask certain problems.

6 Clean & Modern UI
Responsive design for better usability.
